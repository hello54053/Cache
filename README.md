# Cache

**Cache一致性监听协议**

1.一共有4个处理节点，分别为CPU00，CPU01，CPU10，CPU11；

2.一共有4个存储节点，每个4MB，统一编址，采用高位交叉编地址方式；

3. 每个节点的Cache块数为16块，块大小为16B，直接映射；

4. 采用写作废协议和写回法；   

5. 数据提供按照就近原则，即若节点Cache和主存中都有，由Cache提供。

6. 自编仿真程序且有图形化演示界面，能够演示出每个访存请求处理完成后，每个Cache块标记、状态和数据的变化。

**Cache一致性目录协议**

1.一共有4个处理节点，分别为CPU00，CPU01，CPU10，CPU11；

2.每个节点的私有存储器，每个4MB，统一编址，采用高位交叉编地址方式；

3. 每个节点的Cache块数为16块，块大小为16B，直接映射；

4. 采用写作废协议和写回法；  

5. 自编仿真程序且有图形化演示界面，能够演示出每个访存请求处理完成后，每个存储块状态、Cache块标记、状态和数据的变化。
